## 这是什么

手写实现一些JS原生方法和一些工具方法（长期更新）

## 如何使用

本项目使用了ES6模块化，运行`test.html`以进行模块测试

## 模块列表

- `bind.js`：bind方法
- `call.js`：call和apply方法
- `compose.js`：函数式编程中的组合函数
- `create.js`：Object.create方法
- `curry.js`：柯里化函数
- `debounce.js`：防抖
- `deepClone.js`：深拷贝
- `flat.js`：数组扁平化
- `inherit.js`：原型继承
- `instanceof.js`：instanceof的函数实现
- `new.js`：new的函数实现
- `run.js`：自执行生成器
- `throttle.js`：节流
- `transform.js`：3个重要的高阶函数，map、reduce和filter
- `unique.js`：数组去重
